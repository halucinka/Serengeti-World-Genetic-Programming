Main file is DrawWorld.

Description in Slovak:
Po spustení program simuluje pohyb levov a gaziel v Serengetiho svete.
-Vpísaním čísla do políčka "šírka sveta" a "výška sveta" nastavíte rozmery Serengetiho sveta.
-Políčkom "viditeľnosť" nastavujete počet jednotiek, na ktoré vidí gazela aj lev.
-Vpísaním čísla do polícka Počet zač. gaziel" a "počet zač. levov" nastavíte počet začiatočných gaziel alebo počet začiatocných levov.
-Políčkom "počet tráv" nastavíme koľko tráv sa má vygenerovať po vytvorení novej generácie a zároveň políčko "energia tráv" hovorí koľko energie máme rozložiť medzi tieto trávy.
-Políčko "počet jedincov v danej generácií hovorí samo za seba".
Ak je toto všetko nastavené, môžte stlačit Run. Ak sa vám niečo nepáči alebo ak simulácia skončila, opätovne môžete nastavit parametre a zavolat Run.
